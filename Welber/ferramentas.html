Nagios

▼

Requisitos Mínimos: Linux, 1GB RAM, 1 CPU core

Descrição: Nagios é uma ferramenta de monitoramento de rede de código aberto que permite monitorar sistemas, redes e infraestrutura. Ele oferece alertas e relatórios detalhados.

Site Oficial: https://www.nagios.org/

Zabbix

▼

Requisitos Mínimos: Linux, 2GB RAM, 2 CPU cores

Descrição: Zabbix é uma solução de monitoramento de rede e aplicações que oferece monitoramento em tempo real, alertas e visualização de dados.

Site Oficial: https://www.zabbix.com/

Cacti

▼

Requisitos Mínimos: Linux, 1GB RAM, 1 CPU core

Descrição: Cacti é uma ferramenta de monitoramento de rede baseada em RRDtool que permite a criação de gráficos de desempenho de rede.

Site Oficial: https://www.cacti.net/

Icinga

▼

Requisitos Mínimos: Linux, 1GB RAM, 1 CPU core

Descrição: Icinga é um fork do Nagios que oferece uma interface web moderna e recursos adicionais de monitoramento.

Site Oficial: https://icinga.com/
